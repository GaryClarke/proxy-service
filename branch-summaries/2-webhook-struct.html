✅ Branch Summary: 2-add-webhook-model

📌 Purpose

Introduce a typed representation for incoming webhook payloads and metadata to prepare for structured decoding in the handler layer.

🔑 Key Changes

  • Added internal/webhook/webhook.go: 
    package webhook

// Webhook represents an incoming webhook payload with associated metadata.
type Webhook struct {
Payload string `json:"payload"`
Meta    Meta   `json:"meta"`
}

// Meta contains metadata information for a webhook.
type Meta struct {
Client       string `json:"client"`
Notification string `json:"notification"`
}
    Defines:
    • Webhook.Payload: raw JSON payload string.
    • Webhook.Meta: nested Meta struct with Client and Notification fields.
